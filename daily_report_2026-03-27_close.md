## 💡 经验教训

### Cron 环境差异

Cron 的 PATH 环境与交互式 shell 不同：

```bash
# ❌ 可能失败 (依赖 shell 环境)
source venv/bin/activate && python3 script.py

# ✅ 可靠 (绝对路径)
/path/to/venv/bin/python3 script.py
```

### 快速恢复命令

```bash
# 手动执行
cd ~/openclaw/workspace/chanlunInvester
./venv/bin/python3 premarket_report.py  # 盘前
./venv/bin/python3 monitor_all.py       # 监控
```
