## 🔧 手动操作

### 运行一次报告

```bash
cd /home/wei/.openclaw/workspace/trading-system
PYTHONPATH=python-layer:$PYTHONPATH python3 scripts/daily_hot_stocks_report.py
```

### 强制运行 (忽略交易日检查)

```bash
python3 scripts/daily_hot_stocks_report.py --force
```

### 查看日志

```bash
# 查看推送日志
tail -f logs/daily_hot_stocks.log

# 查看历史报告
ls -lht logs/hot_stocks_*.json
```

### 查看 Cron 状态

```bash
crontab -l | grep hot_stocks
```

## 📊 信号强度说明

| 强度范围 | 信号 | 操作 |
|----------|------|------|
| **≥+8.0** | STRONG_BUY++ | 强烈买入++ |
| **≥+6.0** | STRONG_BUY | 强烈买入 |
| **≥+4.0** | BUY | 买入 |
| **-4 ~ +4** | HOLD | 观望 |
| **≤-4.0** | SELL | 卖出 |
| **≤-8.0** | STRONG_SELL | 强烈卖出 |

## ⚠️ 注意事项

### 推送失败处理

如果某天没有收到推送:

1. **检查日志:**
   ```bash
   tail -20 logs/daily_hot_stocks.log
   ```

2. **手动运行:**
   ```bash
   python3 scripts/daily_hot_stocks_report.py --force
   ```

3. **检查 Cron:**
   ```bash
   crontab -l | grep hot_stocks
   ```

### 节假日处理

脚本会自动跳过:
- 周末 (周六、周日)
- 美国法定节假日 (元旦、感恩节、圣诞节等)

### 时区说明

- **推送时间:** 9:00 AM EST (美国东部时间)
- **UTC 时间:** 13:00 UTC
- **北京时间:** 次日 1:00 AM (冬令时) / 次日 12:00 AM (夏令时)

## 🔄 修改推送时间

编辑 Cron:
```bash
crontab -e
```

修改时间 (格式：分 时 日 月 周):
```cron
# 每天早上 9 点 (默认)
0 13 * * 1-5

# 每天早上 8 点
0 12 * * 1-5

# 每天早上 10 点
0 14 * * 1-5
```
