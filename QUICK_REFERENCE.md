## 🔑 关键技术要点

### 新笔定义 (3K 线) ⭐
```
顶分型最高 K 线 和 底分型最低 K 线 之间
→ 不考虑包含关系
→ 至少 3 根 K 线 (包括 3 根)
```

### 线段划分两种情况
```
情况一 (无缺口): 特征序列顶分型中第一二元素无缺口
                 → 线段在顶分型高点结束

情况二 (有缺口): 特征序列顶分型中第一二元素有缺口
                 → 需要后续底分型确认
```

### 三类买卖点
```
第一类：趋势背驰点 (风险最大，收益最大)
第二类：次级别回踩不破前低/高 (风险较小)
第三类：次级别回踩不破中枢 (趋势延续确认)
```
