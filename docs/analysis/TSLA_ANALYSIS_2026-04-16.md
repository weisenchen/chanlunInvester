## 💡 为什么没有推送

### 1. 热点报告有 TSLA

热点股票扫描 (`hot_stocks_resonance_scan.py`) 使用的是**宽松的共振过滤**：

```python
# 热点扫描逻辑
if confidence >= 0.75:  # 置信度≥75%
    ✅ 推送
```

**TSLA 热点报告状态**:
```
🟢 TSLA @ $387.64 (共振 85%)
```

这是基于**多级别走势共振**的评估，不是严格的买卖点定义。

---

### 2. 实时监控无 TSLA

`monitor_all.py` 使用的是**严格的缠论买卖点定义**：

```python
# sell1 条件
if last_high.price > prev_high.price:  # 价格必须新高
    if last_macd < prev_macd:  # MACD 背驰
        ✅ sell1 信号

# sell2 条件
if current_trend == 'down':  # 必须在下跌趋势中
    if last_high.price < prev_high.price:  # 反弹不过前高
        ✅ sell2 信号
```

**TSLA 实时监控状态**:
```
❌ sell1: 价格未新高 ($390.68 < $394.65)
❌ sell2: 当前笔方向向上 (不是下跌趋势)

结果：无信号
```
