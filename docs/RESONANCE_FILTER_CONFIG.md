# 多级别共振过滤配置说明

**更新日期**: 2026-04-10  
**版本**: v5.2 Alpha  
**功能**: 减少警报噪音，只推送高价值入场信号

---

## 🎯 功能概述

启用多级别共振过滤后，系统只在以下条件满足时才发送警报：

1. **多级别共振确认** (优先)
   - 日线 (1d) 和 30 分钟 (30m) 同时出现同向买卖点
   - 例如：1d 买 2 + 30m 买 2 → 共振确认 ✅

2. **高置信度单级别信号** (备选)
   - 无日线信号时，仅推送置信度 ≥75% 的 30m 信号
   - 避免低质量震荡信号干扰

---

## ⚙️ 配置参数

### monitor_all.py

```python
# 多级别共振过滤 (Multi-Level Resonance Filter)
ENABLE_RESONANCE_FILTER = True      # 启用共振过滤
RESONANCE_MIN_CONFIDENCE = 0.75     # 最低置信度要求 (75%)
```

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `ENABLE_RESONANCE_FILTER` | `True` | 是否启用共振过滤 |
| `RESONANCE_MIN_CONFIDENCE` | `0.75` | 单级别信号最低置信度 (0.0-1.0) |

```python
ENABLE_RESONANCE_FILTER = False  # 禁用
```

**效果**: 恢复所有信号推送，噪音较多

## 🎯 使用建议

### 适合人群

- ✅ **忙碌的投资者**: 不想被频繁警报打扰
- ✅ **趋势交易者**: 只关注高置信度入场点
- ✅ **缠论初学者**: 需要明确的共振信号指导

### 不适合人群

- ❌ **高频交易者**: 需要捕捉所有波动
- ❌ **震荡策略**: 需要单级别买卖点信号
- ❌ **研究分析**: 需要完整数据做回测

### 最佳实践

1. **启用共振过滤**: 日常监控使用
2. **禁用过滤**: 回测或研究时临时关闭
3. **调整阈值**: 根据市场波动性调整 `RESONANCE_MIN_CONFIDENCE`
4. **结合其他工具**: 配合 `multi_level_confirmation.py` 使用
