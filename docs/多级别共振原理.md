# 多级别共振原理

**创建日期**: 2026-04-06  
**核心理念**: 只有多级别联动的背驰才会有较高的准确率

---

## 📚 缠论核心原理

> "大级别的背驰，最终都要通过次级别的走势来确认。"
> 
> "单级别背驰可能是中继，多级别共振才是转折。"

---

## 🎯 为什么单级别背驰准确率低？

### 问题 1: 背驰可能是中继

```
单级别背驰（如 30m 顶背驰）:

价格
  │
  │      创新高
  │     ┌──┐
  │     │  │
  │     │  └─┐
  │     │    │  MACD 未创新高（背驰）
  │     │    │
  │     │    └─→ 可能继续上涨 ❌
  │     │
  └─────┴─────────→

结果：背驰后价格继续创新高（背驰失效）
准确率：~50-60%
```

### 问题 2: 强趋势中背驰频繁失效

```
强上涨趋势中的 30m 背驰:

时间：T1    T2    T3    T4
背驰：有    有    有    有
结果：涨    涨    涨    跌 ← 第 4 次才失效

准确率：25%（前 3 次都失败）
```

### 问题 3: 小级别噪音多

```
5m 级别背驰统计（单日）:
- 背驰信号：8-12 次
- 真正转折：2-3 次
- 准确率：~25%
```

## ✅ 多级别共振为什么准确率高？

### 原理 1: 大级别定方向

```
日线（1d）背驰 = 大趋势可能转折
  ↓
30 分钟（30m）背驰 = 中期趋势确认转折
  ↓
5 分钟（5m）背驰 = 短期精确入场点

大级别 = 战略方向
次级别 = 战术执行
```

### 原理 2: 走势类型自相似性

```
缠论核心：走势具有分形自相似性

日线下跌线段 = 30m 下跌走势类型的连接
30m 下跌线段 = 5m 下跌走势类型的连接

当大级别和次级别同时背驰：
- 大级别：趋势动能衰竭
- 次级别：内部结构完成
- 共振：转折概率显著提升
```

### 原理 3: 多重确认过滤假信号

```
单级别背驰 → 50-60% 准确率
  + 次级别背驰确认 → 70-80% 准确率
  + 第二类买卖点确认 → 85-90% 准确率

每多一层确认，准确率提升 15-20%
```

## 🎯 最佳实践

### 1. 耐心等待共振

```
❌ 错误：看到背驰就入场
✅ 正确：等待大级别 + 次级别共振

单级别背驰 → 观察
多级别共振 → 关注
完整确认链 → 入场
```

### 2. 大级别优先

```
优先级：
1d 信号 > 30m 信号 > 5m 信号

1d 背驰 + 30m 确认 = 高胜率
30m 背驰 + 5m 确认 = 中等胜率
5m 单级别 = 低胜率（噪音多）
```

### 3. 方向一致

```
关键检查：
大级别背驰方向 = 次级别背驰方向

1d 底背驰 + 30m 底背驰 = ✅ 共振
1d 底背驰 + 30m 顶背驰 = ❌ 矛盾（不推送）
```

### 4. 超时处理

```
确认窗口内未收到次级别确认：
- 可能背驰失效
- 可能趋势太强
- 重新评估，不盲目入场
```
