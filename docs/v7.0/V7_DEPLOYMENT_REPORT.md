## ⚠️ 已知问题

### 问题 1: SMR 趋势段划分不准确

**现象**: SMR 日线只识别出 1 个趋势段 (实际应该有 2 个)

**原因**: 趋势逆转判断阈值可能过高

**解决**: 调整 `REVERSAL_THRESHOLD` 参数 (当前 5%)

---

### 问题 2: 日线短线参与标的为 0

**现象**: 暂无短线参与标的

**原因**: 当前市场整体处于衰退期

**解决**: 正常现象，等待市场进入成长期/诞生期
