## 📊 五步高置信度交易流程

```
1️⃣ 日线背驰出现     →  方向确立 🎯
     ↓
2️⃣ 背驰段缩量       →  力量衰竭验证 ✅ (volume_confirmation.py)
     ↓
3️⃣ 30m 形成买卖点   →  时机确认 ⏰
     ↓
4️⃣ 确认时放量       →  新资金入场验证 💰 (volume_confirmation.py)
     ↓
5️⃣ 多因素共振入场   →  高置信度执行 🚀 (confidence_calculator.py)
```

## 🎓 核心学习点

### 1. 成交量的核心价值
- **背驰段缩量** = 力量衰竭的直接证据
- **确认段放量** = 新资金入场的确认信号
- **量价背驰** = 高置信度反转信号

### 2. MACD 的多维度应用
- **零轴位置** > 单纯的金叉死叉
- **柱状图面积** > 单纯的柱状图高低
- **多周期共振** > 单周期信号

### 3. 综合可信度的意义
- 单一因素 → 容易被误导
- 多因素共振 → 显著提升可靠性
- 量化评估 → 避免主观判断

---

## 💡 关键洞察

> "缠论的精髓不是预测，而是等待市场自己确认。"

**成交量**和**MACD**的作用不是替代价格结构，而是**验证**价格结构所揭示的趋势变化。

**五步流程**的核心逻辑：
1. 价格结构告诉我们"可能反转"
2. 成交量告诉我们"力量衰竭/入场"
3. MACD 告诉我们"动能变化"
4. 多级别告诉我们"趋势共振"
5. 综合评估告诉我们"是否值得行动"
