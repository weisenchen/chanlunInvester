# 热门股票 + ETF 共振分析报告系统

**更新日期**: 2026-04-10  
**版本**: v5.2 Alpha

## 🎯 功能概述

基于同样的多级别共振过滤条件，扫描热门股票和 ETF 池，生成买卖点分析报告。

### 推送时间

| 报告类型 | 推送时间 (EDT) | 说明 |
|----------|----------------|------|
| **盘前报告** | 09:00 (开盘前 30 分钟) | 预览当日机会 |
| **收盘前报告** | 15:00 (收盘前 1 小时) | 总结当日信号 |

### 扫描范围

| 板块 | 股票数量 | 示例 |
|------|----------|------|
| AI/芯片 | 11 只 | NVDA, AMD, INTC, AVGO |
| 科技巨头 | 9 只 | AAPL, MSFT, GOOGL, META |
| AI 应用 | 8 只 | TSLA, PLTR, SNOW, PANW |
| 半导体设备 | 7 只 | ASML, TSM, AMAT, TEL |
| 金融 | 8 只 | JPM, BAC, WFC, GS |
| 医疗 | 8 只 | JNJ, UNH, PFE, MRK |
| 消费 | 8 只 | WMT, PG, KO, PEP |
| **ETF** | **10 只** | **QQQ, SPY, SOXX, SMH** |
| **总计** | **69 只** | 覆盖主要热门标的 |

## 📊 共振过滤条件

与 `monitor_all.py` 使用相同的过滤逻辑：

### 推送条件 (满足任一)

1. **多级别共振确认** (优先)
   ```
   日线 (1d) 买/卖点 + 30m 同向买/卖点 → ✅ 推送
   置信度提升至 85%
   ```

2. **高置信度单级别信号** (备选)
   ```
   仅 30m 信号，但置信度 ≥75% → ✅ 推送
   ```

### 过滤效果

| 指标 | 数值 |
|------|------|
| 扫描股票数 | 69 只 |
| 预期共振信号 | 0-10 只/日 |
| 过滤率 | ~85-95% |
| 报告质量 | 高置信度机会 |

## 📌 与其他功能集成

### 与 monitor_all.py 协同

- **monitor_all.py**: 实时监控现有持仓 (10 只)
- **hot_stocks_resonance_scan.py**: 扫描全市场机会 (69 只)

**工作流**:
1. 09:00 收到盘前报告 → 了解全市场机会
2. 09:30-16:00 实时监控持仓 → 捕捉买卖点
3. 15:00 收到收盘前报告 → 总结当日信号

### 与 multi_level_confirmation.py 协同

- **multi_level_confirmation.py**: 深度分析特定标的
- **hot_stocks_resonance_scan.py**: 广度扫描全市场

**工作流**:
1. 盘前报告发现机会 → 记录感兴趣的股票
2. 使用 multi_level_confirmation.py 深度分析
3. 决定是否加入监控列表

## ⚠️ 注意事项

### 数据延迟

- Yahoo Finance 数据可能有 15 分钟延迟
- 盘前/盘后数据可能不完整
- 关键决策建议使用实时数据源

### 报告局限性

- 仅基于技术分析 (缠论)
- 不考虑基本面因素
- 不替代个人研究判断

### 使用建议

1. **参考而非依赖**: 报告仅供参考，需结合个人判断
2. **共振优先**: 优先关注多级别共振确认信号
3. **仓位控制**: 即使共振确认，也需控制仓位
4. **止损设置**: 入场前设置好止损位
