## 📈 分析内容

### 多级别监控

**同时监控两个级别:**
- **30 分钟级别** - 主操作级别 (中线)
- **5 分钟级别** - 次级别 (短线/区间套)

**任一级别出现买卖点都会发送提醒！**

### 1. 分型分析 (Fractal)
- 顶分型检测
- 底分型检测
- 包含关系处理

### 2. 笔分析 (Pen)
- 新笔定义 (3 K 线)
- 向上笔 / 向下笔
- 笔的力度

### 3. 线段分析 (Segment)
- 特征序列
- 线段划分 (两种情况)
- 线段方向

### 4. 背驰分析 (Divergence)
- MACD 辅助判断
- 顶背驰 / 底背驰
- 背驰强度计算

### 5. 买卖点识别 (Buy/Sell Points)
- **第一类买卖点:** 趋势背驰点
- **第二类买卖点:** 次级别回踩
- **第三类买卖点:** 中枢突破回踩

## ⚠️ 风险提示

1. **缠论分析仅供参考，不构成投资建议**
2. **UVIX 是高波动性产品，可能不适合所有投资者**
3. **严格执行止损，控制仓位**
4. **市场有风险，投资需谨慎**

### 缠论操作原则 (第 13 课)

- **带套操作:** 分批介入，不一次性满仓
- **严格止损:** 走势从"能搞"变"不能搞"立即退出
- **不预测:** 只按买卖点操作，不臆测走势

## 🔧 自定义配置

编辑 `config/uvix_cron.yaml`:

```yaml
# 修改操作级别
timeframe: 5m    # 5 分钟级别 (更短线)
timeframe: 30m   # 30 分钟级别 (中线)
timeframe: day   # 日线级别 (长线)

# 修改 MACD 参数 (第 24 课)
macd:
  fast: 12
  slow: 26
  signal: 9

# 修改提醒条件
alert:
  triggers:
    on_confidence_above: 0.8  # 置信度高于 80%
```
