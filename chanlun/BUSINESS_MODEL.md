# 缠论 Invester 商业模式设计

**版本:** v1.0  
**日期:** 2026-02-22  
**模式:** Freemium (免费 + 订阅制)

---

## 1. 商业模式概览

### 1.1 核心价值主张

**为投资者提供基于缠论 108 课的 AI 原生分析服务，通过智能化、实时化的买卖点识别，帮助用户提高投资成功率。**

### 1.2 目标客户

| 客户类型 | 特征 | 需求 | 付费意愿 |
|----------|------|------|----------|
| **散户投资者** | 资金<50 万，无编程能力 | 简单易懂的买卖点提示 | 中 |
| **缠论爱好者** | 学习缠论，有一定基础 | 精确的分析工具 | 中高 |
| **量化交易者** | 有编程能力，策略导向 | API 接口、回测功能 | 高 |
| **投资顾问** | 管理多个账户 | 批量分析、报告生成 | 高 |
| **机构投资者** | 资金规模大 | 私有化部署、定制开发 | 极高 |

## 2. 产品分层设计

### 2.1 免费版 (Free Tier)

**定位:** 吸引用户、建立信任、培养习惯

**功能限制:**
- ✅ 基础缠论分析 (分型、笔、线段、中枢)
- ✅ 买卖点识别 (历史数据)
- ⚠️ **预警次数:** 3 次/月
- ⚠️ **监控股票数量:** 最多 5 只
- ⚠️ **数据延迟:** 15 分钟
- ⚠️ **分析周期:** 仅日线、30 分钟
- ❌ 无回测功能
- ❌ 无 API 访问
- ❌ 无多级别联立分析

**获客渠道:**
- GitHub 开源 (核心算法)
- 免费 Web 版 (基础功能)
- 社交媒体内容营销
- 缠论社区推广

### 2.2 专业版 (Pro Tier) - ¥199/月 或 ¥1,999/年

**定位:** 主力收入来源，面向serious投资者

**功能:**
- ✅ 所有免费版功能
- ✅ **预警次数:** 无限
- ✅ **监控股票数量:** 最多 50 只
- ✅ **数据延迟:** 实时
- ✅ **全部分析周期:** 1 分钟至月线
- ✅ **多级别联立分析** (区间套定位)
- ⚠️ **回测功能:** 基础版 (10 次/月)
- ❌ 无 API 访问
- ❌ 无私有化部署

**额外服务:**
- 微信/钉钉/飞书预警通知
- 每日买卖点汇总报告
- 优先技术支持

### 2.3 机构版 (Enterprise Tier) - ¥9,999/月 起

**定位:** 高客单价，面向专业机构

**功能:**
- ✅ 所有专业版功能
- ✅ **监控股票数量:** 无限
- ✅ **API 访问:** 完整 REST + WebSocket API
- ✅ **回测功能:** 高级版 (无限次)
- ✅ **实盘交易接口:** VNPy/MQ 集成
- ✅ **私有化部署:** 本地服务器
- ✅ **定制开发:** 按需求定制
- ✅ **专属技术支持:** 7×24 小时

**定价模式:**
- 基础版: ¥9,999/月 (100 万资金以下)
- 标准版: ¥29,999/月 (1000 万资金以下)
- 定制版: 面议 (1000 万以上)

### 2.4 开发者版 (Developer Tier) - ¥499/月

**定位:** 面向量化开发者、独立交易员

**功能:**
- ✅ 所有专业版功能
- ✅ **API 访问:** REST API (1000 次/天)
- ✅ **MCP 工具访问:** 完整工具集
- ✅ **数据下载:** 历史数据导出
- ✅ **策略回测:** 云端回测环境

```sql
-- 用户订阅表
CREATE TABLE subscriptions (
    id              UUID PRIMARY KEY,

CREATE TABLE usage_quotas (
    id              UUID PRIMARY KEY,
    user_id         UUID NOT NULL REFERENCES users(id),
    quota_type      VARCHAR(50) NOT NULL,  -- bsp_query, alert, api_call, backtest
    used_count      INTEGER NOT NULL DEFAULT 0,
    limit_count     INTEGER NOT NULL,
    reset_date      TIMESTAMP NOT NULL,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- API 调用日志表
CREATE TABLE api_usage_logs (
    id              BIGSERIAL PRIMARY KEY,
    user_id         UUID NOT NULL,
    endpoint        VARCHAR(100) NOT NULL,
    mcp_tool        VARCHAR(100),
    request_time    TIMESTAMP NOT NULL,
    response_time_ms INTEGER,
    status_code     INTEGER,
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_api_usage_user_time ON api_usage_logs(user_id, request_time);
```

### 6.3 配额检查中间件

```python
from functools import wraps
from typing import Dict, Callable

QUOTA_LIMITS = {
    'free': {
        'bsp_query': 3,        # 3 次/月
        'alert_subscription': 5,  # 5 只股票
        'api_call': 1000,      # 1000 次/天
        'backtest': 0,         # 不支持
    },
    'pro': {
        'bsp_query': -1,       # 无限
        'alert_subscription': 50,
        'api_call': 10000,
        'backtest': 10,        # 10 次/月
    },
    'enterprise': {
        'bsp_query': -1,
        'alert_subscription': -1,
        'api_call': -1,
        'backtest': -1,
    }
}

def check_quota(quota_type: str):
    """检查用户配额"""
    def decorator(func: Callable):
        @wraps(func)
        async def wrapper(user_id: str, *args, **kwargs):
            # 获取用户订阅计划
            subscription = await get_user_subscription(user_id)
            plan = subscription.plan
            
            # 获取配额限制
            limit = QUOTA_LIMITS[plan].get(quota_type, 0)
            
            # -1 表示无限
            if limit == -1:
                return await func(user_id, *args, **kwargs)
            
            # 检查已用额度
            used = await get_usage_count(user_id, quota_type)
            if used >= limit:
                raise QuotaExceededError(
                    f"Quota exceeded for {quota_type}. "
                    f"Used: {used}, Limit: {limit}. "
                    f"Please upgrade your plan."
                )
            
            # 执行请求并记录使用
            result = await func(user_id, *args, **kwargs)
            await increment_usage(user_id, quota_type)
            
            return result
        return wrapper
    return decorator

# 使用示例
@check_quota('bsp_query')
async def query_buy_sell_points(user_id: str, symbol: str, level: str):
```
